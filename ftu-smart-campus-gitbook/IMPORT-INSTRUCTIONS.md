# Hướng dẫn nhập bộ tài liệu vào GitBook

## Cách 1: Import ZIP

1. Mở Space **FTU Smart Campus - Hướng dẫn sử dụng**.
2. Chọn **Edit**.
3. Chọn **Add new / Import pages**.
4. Tải file ZIP của bộ tài liệu này lên.
5. Kiểm tra cây điều hướng và ảnh.
6. Chưa publish ngay; thực hiện vòng review nội dung trước.

## Cách 2: Git Sync

1. Giải nén bộ tài liệu.
2. Đưa toàn bộ thư mục vào một GitHub repository.
3. Trong GitBook, thiết lập Git Sync với repository đó.
4. Chọn nhánh dùng để đồng bộ.

## Kiểm tra sau import

- Trang chủ hiển thị đúng.
- Mục lục theo `SUMMARY.md` được nhận diện.
- Tất cả ảnh trong `assets/images` hiển thị.
- Link chéo giữa các trang hoạt động.
- Tiêu đề, callout và bảng không bị lỗi.
- Quyền truy cập Space đang ở chế độ phù hợp trước khi publish.
